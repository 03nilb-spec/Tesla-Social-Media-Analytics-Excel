# Tesla-Social-Media-Analytics-Excel
This project analyzes Tesla's social media activity using Excel to extract insights about engagement trends, mentions, and overall social media performance.


## Project Overview
The goal of this project is to explore Tesla's social media data and provide insights into audience engagement and trends over time. The analysis is done entirely in Excel using pivot tables, charts, and formulas.

## Contents
- *Tesla Social Media Analytics* --> excel file
- *Document Tesla* --> Contains the project report explaining the steps, methodology.

## Key Insights 
- Track Tesla's engagement trends on social media platforms.
- Identify most popular posts and high-performing content types.
- Multi-platform strategy (Instagram, YouTube, FaceBook, Twitter)

## How to Use 
- Open the Excel file *Tesla Social Media Analytics*.
- Refer to the *Document Tesla* for detailed explanations of the analysis steps.
- Explore key insights.

## Tools Used 
- Microsoft Excel
- PDF Editor
